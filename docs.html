Neural Admin Pro

Documentación del producto

Resumen público. La guía completa se entrega dentro del ZIP adquirido.

Qué se entrega

Frontend Angular, API NestJS, worker de correo, PostgreSQL, migraciones, Docker Compose y documentación editable.

Instalación

El camino oficial es Docker Compose: copiar .env.example, levantar los servicios, aplicar migraciones y registrar el propietario de la organización.

Integraciones

Los sistemas externos se conectan por REST y CSV. La API dispone de OpenAPI en /api/docs cuando la instalación está en ejecución.

Soporte y límites

Incluye ayuda de instalación y corrección de bugs reproducibles. No incluye hosting, configuración de proveedores ni integraciones personalizadas.
